# TicTacToe

Mobile-optimized TicTacToe game built with vanilla HTML/CSS/JavaScript.

## Features

- 2-Player mode and unbeatable Computer mode (minimax)
- Mobile-first responsive design
- Animated marks and winning row highlight
- Score persistence via localStorage
- Safe-area insets for notch devices

## Run locally

Just open `index.html` in a browser, or serve it:

```bash
npx serve .
```
